项目介绍：
私人物品智能管理系统  

Python+Flask+MYSQL+DeepSeek

app.py为json形式保存信息
app2.py为数据库保存信息

运行:
1.python app.py
2.python app2.py
static\uploads为保存图片的目录，启动项目添加物品后，图片会保存到这里
